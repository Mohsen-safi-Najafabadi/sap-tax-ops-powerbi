# User Stories

## Simple Framework
As a <role>, I want <goal>, so that <impact>.

---

## Stories (draft)
1) As a Tax Analyst, I want near real-time tax movement visibility, so that I can investigate anomalies quickly.
2) As a Finance Controller, I want standardized KPIs, so that reporting is consistent across teams.
3) As an Ops Manager, I want SLA + volume + backlog in one view, so that I can reduce unresolved cases.
4) As a CX Lead, I want unified quality indicators, so that I can track consistency across channels.
5) As an HR Partner, I want engagement + onboarding + turnover signals, so that interventions can be proactive.
