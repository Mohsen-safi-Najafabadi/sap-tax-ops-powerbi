# SAP Tax & Operations Analytics (Power BI)

This repo demonstrates a governed Power BI solution inspired by:
- SAP-based Tax & Operations reporting modernization
- Standardized KPI model & glossary
- Incremental refresh + near real-time updates (architecture)
- Adoption enablement (onboarding + demos)
- Anomaly detection prototype (STL + rolling z-score approach)
- Unified CX metrics (NPS, SLA, service quality)
- HR analytics (engagement, turnover risk, onboarding)

## Project Structure
- /powerbi        -> PBIX/PBIT files
- /data/sample    -> public sample data (no sensitive info)
- /data/schema    -> table schemas & data dictionary
- /docs/kpi       -> KPI glossary
- /docs/user-stories -> user story framework
- /docs/onboarding -> user onboarding guides
- /scripts        -> data generation / prototype logic
- /assets         -> screenshots for docs

## Governance Notes
- No real company data is included.
- Use sample/masked data only.
